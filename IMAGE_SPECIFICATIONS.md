# Timestable Tutor - Card and Thumbnail Image Specifications

## Image Purpose
Create a visually appealing card/thumbnail image that instantly communicates:
1. Educational focus on timestables
2. AI-powered multi-agent system
3. Kid-friendly and engaging
4. Interactive learning platform

## Recommended Dimensions
- **Card Image**: 1280x720px (16:9 ratio) - For presentations and GitHub
- **Thumbnail**: 640x360px (16:9 ratio) - For smaller displays
- **Social Media**: 1200x630px - For sharing

**Primary Colors**
- Blue: #3B82F6 (Math/Logic)
- Purple: #A855F7 (AI/Technology)
- Pink: #F472B6 (Engagement/Fun)
- Yellow: #FCD34D (Learning/Achievement)

**Supporting Colors**
- White: #FFFFFF (Text, highlights)

- Dark Gray: #374151 (Small text)

## Typography

**Main Title**
- Font: Inter, Poppins, or Montserrat (Bold)
- Size: 64-72px
- Color: White with subtle shadow

**Subtitle**
- Font: Inter, Poppins (Medium)
- Size: 24-28px
- Color: White 80% opacity

**Labels**
- Font: Inter (Regular)
- Size: 14-16px
- Color: White or contrasting
